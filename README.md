# Forge Protocol
#### Description: Forge Protocol is a browser based industrial automation and factory building game that challenges players to design, build, and optimize production networks within a large 300x300 generated world. Starting with a small amount of credits, players can construct extraction drills to harvest valuable ores, use conveyor belts to transport resources across their factory, refine raw materials into more valuable products, and combine processed resources to create advanced goods for profits. The game features multiple resource types with varying values and extraction speeds, encouraging different approaches to factory design and resource management. A real time factory simulation featuring production chains and allowing players to watch materials travel through conveyor networks, processing facilities, and trading hubs. With support for building rotation, factory expansion, camera controls, zooming, and a large open map, Forge Protocol provides a flexible sandbox for experimentation and optimization. The game also includes a polished interface, visual effects, interactive tooltips, notifications, and additional hidden content for players who enjoy discovering secrets. Forge Protocol runs directly in any modern web browser without requiring downloads or installation. Whether you're interested in designing efficient production lines, maximizing profits through automation, or simply experimenting with industrial systems, this simple browser game offers an accessible yet engaging factory building experience. You can play and test the latest version of the game live at **https://forge-protocol.onrender.com/**, where all gameplay is available directly from your browser.
